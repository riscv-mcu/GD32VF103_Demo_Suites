  This demo is based on the GD32VF103C-START-V1.0 board. On GD32VF103C-START Wakeup button key, 
LED1 and GD-Link are included.
  On the board, LED1 is connected to PA7. Download the program to the board, LED1 is 
turned on and off every 1s.
